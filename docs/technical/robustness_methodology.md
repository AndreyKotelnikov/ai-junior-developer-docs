# Методология проверки робастности вторичных гипотез H2b и H4

Документ описывает развёрнутую многометодную проверку робастности вторичных гипотез H2b (синергия RAG × граф) и H4 (паритет on-prem ↔ cloud) факторного эксперимента. Цель — прозрачно зафиксировать пределы вывода по вторичным гипотезам, реализованная мощность двустороннего критерия Уилкоксона по которым ниже 0,80.

## Принцип честной отчётности

Для вторичных гипотез применена многометодная проверка робастности на тех же 675 наблюдениях (15 задач × 5 повторов × 9 стендов). Все результаты, включая неблагоприятные, изложены прозрачно в соответствии с принципом честной отчётности (Amrhein, Greenland, McShane 2019).

## H2b — проверка тремя методами

Эффект взаимодействия RAG × граф (синергия): Interaction = (Pass@1(E) − Pass@1(D)) − (Pass@1(G) − Pass@1(B)) > +5 п.п.

| Метод | Результат | Формальная мера |
|---|---|---|
| Wilcoxon (per-task контраст) на 15 разностях | interaction = +9,33 п.п.; *p* = 0,3105 | точечная оценка превышает порог +5 п.п. |
| Bootstrap (10 000 ресэмплов; Efron, 1979) | P(эффект > 0) = 0,845; P(эффект > +5 п.п.) = 0,692 | прямая вероятность достоверности направления |
| Стратификация (3 страты сложности) | простые −4,0; средние +8,0; сложные +24,0 п.п. | монотонный рост со сложностью — согласие с априорным прогнозом |

**Вывод по H2b.** Точечная оценка +9,33 п.п. превышает порог +5 п.п.; направленная поддержка (bootstrap-вероятность 0,845, монотонная дозозависимость) содержательна. Вердикт — «подтверждена условно по точечной оценке»; для строгого формального подтверждения требуется расширение выборки до *n* ≈ 184.

## H4 — TOST и триангуляция

H4 — гипотеза эквивалентности (паритета), для которой двусторонний тест разности методологически неполон. Применён TOST (Two One-Sided Tests; Schuirmann, 1987; Lakens, 2017): нижняя граница одностороннего 95 % ДИ на отношение паритета вычислена bootstrap-аппроксимацией.

Декомпозиция проверки H4:

| Метод | Результат | Критерий подтверждения | Выполнен |
|---|---|---|:-:|
| TOST — non-inferiority на уровне 0,65 | нижняя 95 % одностор. ДИ 0,6575 > 0,65 | формально установлено | ✓ |
| Триангуляция метрик (Pass@1 84,85 %; Completeness 82,56 %; TimeToSuccess 89,1 %) | 3 из 3 > 75 % | 3 / 3 | ✓ |
| Стратификация по 3 стратам (92,0 / 82,6 / 77,8 %) | 3 из 3 > 75 % | 3 / 3 | ✓ |
| Внешний эталон SWE-Bench Pro 14B | +34,85 п.п. над диапазоном 30–50 % | > верхней границы | ◐ (частично, по порядку величины) |
| Bootstrap P(паритет ≥ 0,75) | 0,806 | прямая мера достоверности | ◐ |
| Парный Уилкоксон (комплементарный) | *p* = 0,2008 | согласуется с гипотезой паритета | ◐ |
| TOST — паритет на целевом уровне 0,75 | нижняя 95 % одностор. ДИ 0,6575 | > 0,75 | ✗ |

**Вывод по H4.** Метод TOST формально устанавливает non-inferiority локальной модели на уровне 0,65 (нижняя граница ДИ 0,6575 > 0,65); формального паритета именно на целевом уровне 0,75 на выборке *n* = 15 недостаточно (0,6575 < 0,75). Триангуляция трёх метрик и стратификация трёх страт поддерживают вердикт. Вердикт — «подтверждена условно по точечной оценке».
